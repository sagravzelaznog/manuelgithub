Configuración Inicial

Instalar Git

Descarga e instala Git desde git-scm.com

Configuración Básica

# Configurar nombre de usuario
git config --global user.name "Tu Nombre"

# Configurar correo electrónico
git config --global user.email "tu@email.com"

# Verificar configuración
git config --list

Comandos Básicos

Inicializar un repositorio

# Inicializar repositorio local
git init

Verificar estado

git status

Añadir archivos

# Añadir un archivo específico
git add nombre-archivo.ext

# Añadir todos los archivos
git add .

Hacer commit

git commit -m "Mensaje descriptivo del cambio"

Trabajando con Ramas

Crear y cambiar de rama

# Crear nueva rama
git branch nombre-rama

# Cambiar a la rama
git checkout nombre-rama

# Crear y cambiar de rama en un solo comando
git checkout -b nombre-rama

Fusionar ramas

# Cambiar a la rama principal
git checkout main

# Fusionar la rama
git merge nombre-rama

Trabajando con GitHub

Conectar repositorio local con GitHub

# Añadir repositorio remoto
git remote add origin https://github.com/usuario/repositorio.git

# Verificar repositorios remotos
git remote -v

Subir cambios a GitHub

# Subir rama principal
git push -u origin main

# Para ramas adicionales
git push -u origin nombre-rama

Actualizar repositorio local

git pull origin main

Flujo de Trabajo Básico

  1. Actualiza tu repositorio local

    git pull origin main

  2. Crea una nueva rama

    git checkout -b nombre-caracteristica

  3. Realiza tus cambios

    Edita, añade o elimina archivos según sea necesario.

  4. Añade los cambios

    git add .

  5. Haz commit de los cambios

    git commit -m "Descripción de los cambios"

  6. Sube los cambios

    git push -u origin nombre-caracteristica

  7. Crea un Pull Request

    Ve a GitHub y crea un nuevo Pull Request para fusionar tus cambios.
