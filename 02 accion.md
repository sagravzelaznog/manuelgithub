# accion.md

- Crea un mini curso de github para que puedas aprender a usar github de manera rapida y eficiente.
- Explica y muestra ejemplos paso a paso de los atajos y trucos de github para que puedas crear repositorios y subir archivos de manera rapida y eficiente. crea comentarios en el codigo explicando cada paso.
- Escribe el curso en un archivo md llamado "curso-github.md"
- Explica como crear un repositorio en github y subir archivos de manera rapida y eficiente. con los comandos de git aplicados de forma literal.
- Crea el curso en un archivo html llamado "curso-github.html"
- Crea los estilos  en un archivo css llamado "style.css" con temas de color entre azul y esmeralda.
- Crea la interactividad del curso en un archivo js llamado "script.js"
- Vincula a el archivo html los estilos y la interactividad.
- Crea los derechos de autor en el archivo html. agrega un footer con el contenido "@copy; JMGV-PTEL-2025 Todos los derechos reservados".
- Crea el archivo README.md con la informacion del curso.
- Crea el archivo .gitignore con la informacion del curso.
